# FlowTables
in development

## Goal
A collaboration focused system for primarily data tables but also other forms of notekeeping and data storage.
Its main goal is to provide easy to use tables shared among a team and guests (users without account)
Features may include but are not limited to:
- "Projects" as collection of tables
- Real time entry updates
- Designated table areas for different people/teams
- Long range bluetooth connection between devices to work without internet connection
- Connection between tables

This will be available as App and as Website for easy access.
